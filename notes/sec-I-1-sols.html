    1. Why? Traditional Calculus vs. Math 156.

    Solution.
    Traditional calculus courses typically emphasize algebra/trigonometry, an introduction to limits (infinity and infinitesimals), and derivatives and integrals (tools for measuring change). Ultimately the goal is to be able to approach differential equations, which help us model physical applications.
    🔗
    In math 156, we emphasize logic/relations, functions, derivatives and integrals (with applications), and sequences/series. The goal is to emphasize tools that help us program and compute.

    🔗
    Solution.
    Sets are one of the most basic structures that we use in mathematics, and they’re used often in building other objects and in writing algorithms.

    (a)

    Do you think any sets are the same? Explain your reasoning.
    🔗
    Solution.
    Sets \(A\) and \(D\) are the same, and sets \(B\) and \(E\) are the same. Two sets are equal if they contain the same elements (order and repetition are disregarded).

    1. \(\displaystyle 3 \in A\)

      🔗

    5. Activity: Set Builder Notation.

    For large or more complicated sets, it’s not always convenient (or possible) to write out every element of the set as we did previously. To communicate these sets, we can use set builder notation. The following are examples of sets using set builder notation. Use them to answer the questions.
    1. \(\displaystyle S_1=\{x \in A | x >2\}\)
      🔗
      🔗
    2. \(\displaystyle S_2=\{x \in C | x \text{ is in the Mountain Time Zone}\}\)
      🔗
      🔗
    3. \(\displaystyle S_3=\{x| x=2n \text{ for some integer } n \gt 0\}\)
      🔗
      🔗
    4. \(\displaystyle S_4=\{2x | x \text{ is a positive integer} \}\)
      🔗
      🔗
    5. \(\displaystyle S_5=\{x^2 | x \in B \}\)
      🔗
      🔗
    🔗

    (a)

    It is true that \(S_1 = \{3,4\}\text{.}\) Write an English sentence that describes set builder notation for \(S_1\text{.}\)
    🔗
    Solution.
    The set \(S_1\) is all elements \(x\) in \(A\) such that \(x\) is greater than two. The symbol "\(|\)" reads "such that". It is also common to use a colon ("\(:\)") to indicate the same thing. The left hand side of set builder notation indicates a variable for elements in the set, while the right hand side (called the predicate) indicates a condition that must be met to be in the set.
    🔗
    🔗
    🔗

    (b)

    For \(S_2, S_3, S_4, S_5\text{,}\) write out the sets explicitly, or describe what they are in more common language. Are any sets equal?
    🔗
    Solution.
    1. \(\displaystyle S_2= \{\text{CO}\}\)
      🔗
      🔗
    2. \(S_3\) is the set of all positive even integers.
      🔗
      🔗
    3. \(S_4\) is also the set of all positive even integers.
      🔗
      🔗
    4. \(\displaystyle S_5= \{4, 16, 36\}\)
      🔗
      🔗
    🔗
    🔗
    🔗

    (c)

    Determine if the following are TRUE or FALSE.
    1. \(\displaystyle \{x \in S_3 | x \lt 10\}= \{2, 4, 6, 8\}\)
      🔗
      🔗
    2. \(\displaystyle |\{x | x \text{ is an integer and } |x| \lt 2\}|= \{3\}\)
      🔗
      🔗
    🔗
    Solution.
    1. TRUE
      🔗
      🔗
    2. FALSE. The left hand side indicate the cardinality of a set, which should be a number, not a set. To make this true, replace \(\{3\}\) with \(3\text{.}\)
      🔗
      🔗
    🔗
