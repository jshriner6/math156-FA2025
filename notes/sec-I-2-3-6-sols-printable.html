    1. Motivation: Sets from Sets.

    There are many useful ways to obtain sets from other sets. Intuitively, we may want to:
    🔗
    Solution.
    1. consider only part of a set (subset).
      🔗
      🔗
    2. "add", "subtract", or find common elements of two sets (union, difference, intersection).
      🔗
      🔗
    3. specify an ordering in a set, which is useful in "for" loops, or specify multiplicities of elements in a set (Cartesian product, multiset).
      🔗
      🔗
    🔗
    🔗
    🔗

    2. Definition: Subset.

    Let \(A\) and \(B\) be sets. \(A\) is a subset of \(B\) if
    🔗
    Solution.
    every element of \(A\) is also an element of \(B\text{.}\) We write \(A \subset B\text{.}\)
    🔗
    🔗
    🔗

    3. Example: Subsets.

    Let \(B=\{1,2,3\}\text{.}\) What are all of the subsets of \(B\text{?}\)
    🔗
    Solution.
    All of the subsets are \(\{1\}\text{,}\) \(\{2\}\text{,}\) \(\{3\}\text{,}\) \(\{1,2\}\text{,}\) \(\{1,3\}\text{,}\) \(\{2,3\}\text{,}\) \(\{1,2,3\}\text{,}\) \(\{ \}\) (also written \(\emptyset\) and called the empty set). Note that the full set and the empty set are always subsets. See the text for the Hasse diagram illustrating all subsets of \(B\text{.}\)
    🔗
    🔗
    🔗

    4. Useful Fact: Subsets and Equality.

    🔗
    Solution.
    Subsets can be useful to show equality. \(A=B\) exactly when \(A \subset B\) and \(B \subset A\text{.}\)
    🔗
    🔗
    🔗

    6. Definition: Power Set.

    Let \(S\) be a set. The power set of \(S\) is
    🔗
    Solution.
    the set \(P(S)= \{A | A \subset S \}\text{.}\)
    🔗
    🔗
    🔗

    7. Example: Power Set.

    \(P(\{0,1\}) = \)
    🔗
    Solution.
    \(\{ \{0\}, \{1\}, \{0,1\}, \emptyset \}\)
    🔗
    🔗
    🔗

    13. Note: Universal Sets and Complements.

    Solution.
    If \(A \subset U\text{,}\) the difference \(U \setminus A\) is more commonly written as \(A^C\) (read "\(A\) complement"). \(U\) is called a universal set, which is an ambient space that all of our sets are understood to live in.
    🔗
    For example, suppose our universal set is \(\R\text{.}\) Then \(\Q^C = \{\text{irrational numbers}\}\text{.}\)
    🔗
    🔗
    🔗

    16. Ordered Pairs.

    🔗
    Solution.
    We can specify an ordering with sets using ordered pairs. Recall
    \begin{equation*} \{1,2\} = \{2,1\}\text{.} \end{equation*}
    If we actually want to communicate "1 in the first position and 2 in the second position", we can write \(\{1, \{1,2\}\}\) as a set, which is more commonly written as the ordered pair \((1,2)\text{.}\) Parentheses imply order (e.g., \((x,y)\) in the Cartesian plane).
    🔗
    🔗
    🔗

    17. Definition: Cartesian Product.

    Given sets \(A\) and \(B\text{,}\) the Cartesian product of \(A\) and \(B\) is
    🔗
    Solution.
    \(A \times B = \{(a,b) | a \in A, b \in B\}\text{.}\)
    🔗
    🔗
    🔗

    18. Example: Cartesian Product.

    \begin{equation*} A \times B = \end{equation*}
    🔗
    Solution.
    \(\{(1,4), (1,5), (2,4), (2,5), (3,4), (3,5)\}\text{.}\)

    20. Tuples and Index sets.

    🔗
    Solution.
    The concept of ordered pairs can be extended to tuples, which is can be an ordering of more than two things. For example, \((1,1,2)\) is a \(3\)-tuple. This could represent a point in 3-dimensional space, where the first coordinate specifies a position in the x-direction, the second coordinate specifies a position in the y-direction, and the third coordinate specifies a position in the z-direction.
    🔗
    In programming, we refer to specific elements of a tuple with an index set. For example, in the tuple \(t= (4, -1,-2,1,0)\text{,}\) we can specify elements as \(t_i\) for \(i\) in the index set \(\{1,2,3,4,5\}\text{.}\) More specifically, \(t_1 = 4\text{,}\) \(t_2 = -1\text{,}\) \(t_3 = -2\text{,}\) \(t_4 = 1\text{,}\) and \(t_5 = 0\text{.}\)
    🔗
    Index sets can also be tuples. If we write that \(t = t_{ij}\) for \((i,j) \in \{1,2\} \times \{1,2,3\}\text{,}\) then \(t\) is more commonly visualized as the matrix
    \begin{equation*} \begin{bmatrix} t_{11} & t_{12} & t_{13} \\ t_{21} & t_{22} & t_{23} \end{bmatrix} \end{equation*}
    🔗
    🔗
    🔗

    21. Multisets.

    🔗
    Solution.
    We can specify multiplicities with a multiset. Recall that
    \begin{equation*} \{1,2,2,4,4,4,0\} = \{0,1,2,4\}\text{.} \end{equation*}
    We would capture the multiplicities in the left side by defining
    \begin{equation*} S = \{0, 1,2,4\}\text{,} \end{equation*}
    which is the reduced set and will be used as an index set, and defining the counting set
    \begin{equation*} C= \{c_s | s \in S, c_s \text{ is the multiplicity of } s\}\text{.} \end{equation*}
    Note that in our example, \(C=\{c_0=1, c_1=1, c_2=2, c_4=3\}\text{.}\) The pair \((S,C)\) is considered a multiset.
