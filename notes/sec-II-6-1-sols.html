    2. Fact: Euclidâ€™s Division Lemma.

    For \(a,m \in \Z\) with \(m \neq 0\) (we will take \(m \gt 1\)), there is a unique \(q,r \in \Z\) with \(0 \leq r \lt m\) such that
    \begin{equation*} a = qm +r\text{.} \end{equation*}
    ðŸ”—
    Solution.
    The intuition is if we divide \(a\) by \(m\text{,}\) there is a unique quotient \(q\) and remainder \(r\text{.}\)

    3. Definition: Modulo Equivalence Relation.

    Fix \(m \in \Z\) with \(m \gt 1\text{.}\) The relation \(R_m\) on \(\Z \times \Z \) is defined as
    ðŸ”—
    Solution.
    \(aR_m b\) if and only if \(a\) and \(b\) have the same remainder \(r\) when divided by \(m\text{.}\)
    ðŸ”—
    An equivalent (and many times useful) way to rephrase this is that \(aR_m b\) if and only if \(m\) divides \(a-b\text{.}\)
    ðŸ”—
    Using the definition by remainders it is straightforward to verify that \(R_m\) is reflexive, symmetric, and transitive, and therefore an equivalence relation. Equivalence classes in this context are typically referred to as congruence classes. If \(a\) and \(b\) are in the same congruence class we say that \(a\) and \(b\) are congruent modulo \(m\) (or congruent mod \(m\) for short).
