    Worksheet \(\S III.4\text{:}\) Function Properties
    

    1. Motivation: Function Properties.

    🔗
    Solution.
    When discussing relations, we also defined the complement and converse of a relation. In the context of relations that are functions, the complement relation is rarely a function (why?). However, the converse relation is more interesting. If we start with a function relation, can we swap the domain and range and again obtain a new function relation? To answer this question, we need to discuss two important properties of functions.
    🔗
    🔗
    🔗

    2. Definition: Onto (Surjective).

    🔗
    Solution.
    A function \(f: A \to B\) is onto if for all \(b \in B\) there exists \(a \in A\) such that \(f(a) =b\text{.}\)
    🔗
    For example, \(f: \R \to \R_{\geq 0}\text{,}\) \(x \mapsto x^2\) is onto, but \(f: \R \to \R\text{,}\) \(x \mapsto x^2\) is not onto.
    🔗
    🔗
    🔗

    3. Why?

    🔗
    Solution.
    If \(f\) is onto, we can swap source and target and the domain of the new relation is the whole source, which is a requirement of a function relation.
    🔗
    🔗
    🔗

    4. Practice: Onto.

    Determine if the following functions are surjective:
    1. \(f: \R \to \R\text{,}\) \(x \mapsto 2x\)
      🔗
      🔗
    2. \(g: \Z \to \Z\text{,}\) \(n \mapsto 2n\)
      🔗
      🔗
    3. \(h: \Z / 6\Z \to \Z / 6\Z\text{,}\) \(x \mapsto \overline{3}x\)
      🔗
      🔗
    🔗
    Solution.
    1. \(f\) is onto. If \(b\) is an element in the target, then the element \(x=0.5b\) in the domain is mapped to \(b\text{.}\) How can you tell visually that \(f\) is onto?
      🔗
      🔗
    2. \(g\) is not onto. For example, \(3\) is in the target, but there is no integer in the domain that maps to \(3\text{.}\) How can you tell visually that \(g\) is not onto?
      🔗
      🔗
    3. \(h\) is not onto. For example, \(\overline{1}\) is in the target, but there is no domain element which maps to \(\overline{1}\) (i.e, \(\overline{3}\) has no multiplicative inverse).
      🔗
      🔗
    🔗
    🔗
    🔗

    5. Definition: One-to-one (Injective).

    🔗
    Solution.
    A function \(f: A \to B\) is one-to-one if \(f(a_1)=f(a_2)\) implies \(a_1=a_2\text{.}\) For example, \(f:\R_{\geq 0} \to \R\text{,}\) \(x \mapsto x^2\) is one-to-one, but \(f:\R \to \R\text{,}\) \(x \mapsto x^2\) is not one-to-one.
    🔗
    🔗
    🔗

    6. Why?

    🔗
    Solution.
    If \(f\) is one-to-one, it guarantees the domain of the converse relation will map to unique elements of the range.
    🔗
    Notice that we need \(f\) to be both onto and one-to-one in order for its converse relation to also be a function relation!
    🔗
    🔗
    🔗

    7. Practice: One-to-one.

    Determine if the following functions are injective:
    1. \(f: \Z \to \Z\text{,}\) \(n \mapsto 2n\)
      🔗
      🔗
    2. \(g: \Z / 6\Z \to \Z / 6\Z\text{,}\) \(x \mapsto \overline{3}x\)
      🔗
      🔗
    🔗
    Solution.
    1. \(f\) is one-to-one. If \(2a_1 = 2a_2\text{,}\) then \(a_1 = a_2\text{.}\) How can you tell v isually that \(f\) is one-to-one?
      🔗
      🔗
    2. \(g\) is not one-to-one. For example, \(g(\overline{0}) = g(\overline{2})\text{,}\) but \(\overline{0} \neq \overline{2}\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗

    8. Fact.

    If \(A\) and \(B\) are finite with \(|A| = |B|\text{,}\) then
    🔗
    Solution.
    \(f: A \to B\) is one-to-one if and only if it is onto.
